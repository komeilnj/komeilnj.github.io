# Komeil Najafi Research Portfolio

A static multi page portfolio designed for GitHub Pages.

## Pages

* `index.html` provides a quick overview and profile introduction
* `research.html` summarizes research themes and methods
* `transcript.html` includes official bachelor and master transcript documents, degree summaries, academic context, and verified coursework
* `laboratory.html` contains five expandable project sections with laboratory images and the propagation loss GUI
* `thesis.html` presents the manuscript draft, sensing method, experimental link, and research team
* `international.html` presents Nordita and ICPS experience with supporting images and certificates

## Publish with GitHub Pages

1. Create a repository and upload the contents of this folder to the repository root.
2. Open repository Settings, then Pages.
3. Choose Deploy from a branch.
4. Select the main branch and the root folder.
5. Save and wait for the public address to appear.

The site uses only HTML, CSS, JavaScript, images, and PDF documents. No build step is required.
